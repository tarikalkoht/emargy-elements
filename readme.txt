Emargy Elements adds premium quality widgets to Elementor page builder including the dynamic Timeline Slider widget.

== Description ==

Emargy Elements enhances Elementor with high-quality, responsive widgets that can be seamlessly integrated into your Elementor-powered website. 

The main widget included is the **Timeline Slider**, a dynamic and interactive slider that allows you to showcase your posts, projects, or any custom post type in a unique timeline format.

**Features:**

* Dynamic content loading from any post type
* Smooth animations and transitions
* Interactive timeline with draggable control
* Fully responsive design
* Customizable colors, sizes, and styling options
* Autoplay functionality with speed control

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/emargy-elements` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the widgets in Elementor page builder

== Frequently Asked Questions ==

= Does this plugin require Elementor? =

Yes, Emargy Elements is an addon for Elementor and requires Elementor to be installed and activated.

= Is Elementor Pro required? =

No, the basic functionality works with the free version of Elementor. However, some advanced query features are enhanced if Elementor Pro is installed.
